Exercícios de HTML

1. Qual tag é usada para criar um link em HTML?

  • <link>
  • <href>
  • <url>
2. Qual atributo é OBRIGATÓRIO na tag <img>?

  • alt
  • srcset
  • loading

Desafio Prático:

Crie uma lista ordenada com 3 itens usando HTML:

<ol> <li>Primeiro item</li> <li>Segundo item</li> <li>Terceiro item</li> </ol>

Exercícios de CSS

1. Qual propriedade é usada para alterar a cor do texto?
  • text-color
  • color
  • font-color
  • text-style
2. Como centralizar um elemento horizontalmente com margin?
  • margin: 0 auto;
  • margin: auto 0;
  • margin: center;
  • align: center;

Desafio Prático:

Crie um botão estilizado com CSS que mude de cor ao passar o mouse:

.btn { background-color: #3498db; color: white; padding: 10px 20px; border: none; border-radius: 5px; cursor: pointer; } .btn:hover { background-color: #2980b9; }
