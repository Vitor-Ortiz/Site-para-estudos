HTML - Estrutura da Web

O que é HTML?

Linguagem de Marcação de Hipertexto

HTML (HyperText Markup Language) é a linguagem fundamental para criar páginas web. Ela define a estrutura e o significado do conteúdo.

Exemplo Básico:

<!-- Estrutura básica de um documento HTML -->
<!DOCTYPE html>
<html lang="pt-br">
<head>
  <meta charset="UTF-8">
  <title>Título da Página</title>
</head>
<body>
  <!-- Conteúdo visível da página -->
  <h1>Meu Primeiro Título</h1>
  <p>Meu primeiro parágrafo.</p>
</body>
</html>

Meu primeiro parágrafo.

Elementos HTML

Principais Tags HTML

HTML utiliza "tags" para estruturar o conteúdo. Cada tag tem uma função específica.

Tags Comuns:

<!-- Títulos -->
<h1>Título Principal</h1>
<h2>Subtítulo</h2>
<!-- Parágrafo -->
<p>Este é um parágrafo.</p>
<!-- Link -->
<a href="https://exemplo.com">Link para exemplo</a>
<!-- Imagem -->
<img src="imagem.jpg" alt="Descrição">
Formulários HTML

Coletando Dados do Usuário

Formulários permitem que os usuários enviem dados para o servidor.

Exemplo de Formulário:

<form action="/enviar" method="post">
  <label for="nome">Nome:</label>
  <input type="text" id="nome" name="nome">

  <label for="email">Email:</label>
  <input type="email" id="email" name="email">

  <input type="submit" value="Enviar">
</form>






CSS - Estilização da Web

O que é CSS?

Folhas de Estilo em Cascata

CSS (Cascading Style Sheets) é usado para estilizar e layout páginas web - controlando cores, fontes, espaçamento e posicionamento.

Sintaxe Básica:

/* Seletor { propriedade: valor; } */
body {
  background-color: #f0f0f0;
  font-family: Arial, sans-serif;
  margin: 0;
}

h1 {
  color: blue;
  text-align: center;
}
Seletores CSS

Como Aplicar Estilos

Seletores determinam quais elementos HTML receberão os estilos definidos.

Tipos de Seletores:

/* Seletor de elemento */
p { color: red; }

/* Seletor de classe */
.destaque { background-color: yellow; }

/* Seletor de ID */
#cabecalho { font-size: 24px; }

/* Seletor de atributo */
input[type="text"] { border: 1px solid #ccc; }
Flexbox e Grid

Layouts Modernos

Flexbox e CSS Grid são sistemas de layout que facilitam a criação de designs responsivos.

Exemplo Flexbox:

.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
}

.item {
  flex: 1;
  margin: 10px;
}

JavaScript - Interatividade da Web

O que é JavaScript?

Linguagem de Programação para a Web

JavaScript permite adicionar interatividade, animações e comportamentos dinâmicos às páginas web.

Primeiro Código JavaScript:

// Exibe um alerta no navegador
alert('Olá, Mundo!');

// Escreve no console (F12 para ver)
console.log('Mensagem no console');

// Altera o conteúdo de um elemento HTML
document.getElementById('meuElemento').innerHTML = 'Novo conteúdo';
Variáveis e Funções

Conceitos Fundamentais

Variáveis armazenam dados e funções executam ações específicas.

Declarando Variáveis e Funções:

// Declarando variáveis
let nome = 'João'; // Variável que pode ser reatribuída
const idade = 25; // Constante (não pode ser reatribuída)

// Declarando uma função
function cumprimentar() {
  return 'Olá, ' + nome + '! Você tem ' + idade + ' anos.';
}

// Chamando a função
console.log(cumprimentar());

Manipulação do DOM

Interagindo com a Página

O DOM (Document Object Model) permite que o JavaScript altere o conteúdo e estilo dos elementos HTML.

Alterando Elementos:

// Alterar texto de um elemento
document.getElementById('meuTexto').textContent = 'Novo texto';

// Alterar estilo de um elemento
document.getElementById('meuElemento').style.color = 'red';

// Adicionar/remover classes
document.getElementById('meuElemento').classList.add('destaque');
document.getElementById('meuElemento').classList.remove('destaque');
Este é um elemento de demonstração.

Estrutura Básica

<!DOCTYPE html>
Declara o tipo de documento e versão do HTML.
<!DOCTYPE html>
Exemplo: Primeira linha de qualquer documento HTML5.
Declaração Obrigatório
<html>
Elemento raiz que contém todo o conteúdo HTML.
<html lang="pt-br">...</html>
Exemplo: Envolve todo o conteúdo da página.
Raiz Estrutura
<head>
Contém metadados, links para CSS, título da página, etc.
<head>...</head>
Exemplo: Inclui título, charset, viewport e links.
Metadados Cabeçalho

Elementos de Texto

<h1> a <h6>
Cabeçalhos ou títulos, onde h1 é o mais importante.
<h1>Título</h1>
Exemplo: <h1>Título Principal</h1>
Título Hierarquia
<p>
Define um parágrafo de texto.
<p>Texto do parágrafo</p>
Exemplo: <p>Este é um parágrafo de exemplo.</p>
Texto Parágrafo
<span>
Container inline para estilizar parte de um texto.
<span class="destaque">texto</span>
Exemplo: <p>Texto com <span style="color:red;">destaque</span>.</p>
Inline Estilo

Formulários

<form>
Container para elementos de entrada de dados.
<form action="/enviar" method="post">...</form>

<input type="text" name="campo" placeholder="Texto">

Entrada Formulário

Layout e Posicionamento

display
Define como um elemento é exibido na página.
display: block | inline | inline-block | flex | grid | none;
Exemplo: .container { display: flex; }
Uso: Controla o comportamento de layout do elemento.
Layout Fundamental
position
Define o método de posicionamento de um elemento.
position: static | relative | absolute | fixed | sticky;
Exemplo: .menu { position: fixed; top: 0; }
Uso: Para elementos que precisam de posicionamento específico.
Posicionamento Layout
flexbox
Sistema de layout unidimensional para organizar itens.
display: flex; flex-direction: row; justify-content: center;
Exemplo: .container { display: flex; justify-content: space-between; }
Uso: Para layouts responsivos e alinhamento fácil de elementos.
Layout Responsivo

Estilização Visual

box-shadow
Adiciona sombra a um elemento.
box-shadow: offset-x offset-y blur-radius color;
Exemplo: box-shadow: 0 4px 8px rgba(0,0,0,0.1);
Uso: Para criar profundidade e destaque visual.
Efeito Visual
border-radius
Arredonda os cantos de um elemento.
border-radius: 10px;
Exemplo: border-radius: 50%; /* Cria um círculo */
Uso: Para suavizar a aparência de elementos.
Visual Forma
gradient
Cria transições suaves entre cores.
background: linear-gradient(direction, color1, color2);
Exemplo: background: linear-gradient(135deg, #667eea, #764ba2);
Uso: Para fundos modernos e visualmente atraentes.
Cor Fundo

Manipulação do DOM

document.querySelector()
Seleciona o primeiro elemento que corresponde ao seletor CSS.
document.querySelector('.classe ou #id ou tag');
Exemplo: const elemento = document.querySelector('#meuId');
Uso: Para selecionar e manipular elementos específicos da página.
DOM Seleção
addEventListener()
Anexa um manipulador de evento a um elemento.
elemento.addEventListener('evento', função);
Exemplo: botao.addEventListener('click', function() { alert('Clicado!'); });
Uso: Para responder a interações do usuário (cliques, teclas, etc).
Eventos Interação
classList
Manipula as classes CSS de um elemento.
elemento.classList.add() / .remove() / .toggle()
Exemplo: elemento.classList.add('ativo'); elemento.classList.remove('inativo');
Uso: Para alterar estilos dinamicamente através de classes CSS.
Classes Estilo

Arrays e Objetos

map()
Cria um novo array com os resultados de uma função aplicada a cada elemento.
array.map(função);
Exemplo: const dobros = numeros.map(num => num * 2);
Uso: Para transformar dados de um array de forma eficiente.
Array Transformação
filter()
Cria um novo array com elementos que passam em um teste.
array.filter(função);
Exemplo: const maiores = numeros.filter(num => num > 10);
Uso: Para filtrar dados baseado em condições específicas.
Array Filtro
reduce()
Executa uma função redutora em cada elemento, resultando em um único valor.
array.reduce(função, valorInicial);
Exemplo: const soma = numeros.reduce((acc, num) => acc + num, 0);
Uso: Para agregar dados (soma, média, etc) ou transformar arrays em objetos.
Array Agregação
