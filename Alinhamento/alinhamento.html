Teoria de Alinhamento CSS

Entenda os conceitos fundamentais para posicionar e alinhar elementos na web.

1. Propriedade display

O que é: Define como um elemento se comporta no fluxo do documento.

Valores Principais:

display: block; - Ocupa toda a largura, quebra linha
display: inline; - Ocupa só o necessário, não quebra linha
display: inline-block; - Mescla características de ambos
display: flex; - Container flexível (moderno)
display: grid; - Layout em grade (moderno)
display: none; - Esconde o elemento

2. Propriedade position

O que é: Define o método de posicionamento de um elemento.

Valores e Comportamentos:

position: static; - Padrão, segue o fluxo normal
position: relative; - Relativo à sua posição normal
position: absolute; - Relativo ao ancestral posicionado mais próximo
position: fixed; - Relativo à viewport (fica fixo)
position: sticky; - Alterna entre relative e fixed

3. Flexbox - Alinhamento Moderno

O que é: Sistema unidimensional para layout de itens em linha ou coluna.

Container Flex (pai):

display: flex; - Ativa o flexbox
flex-direction: row | column; - Direção dos itens
justify-content: center | flex-start | flex-end | space-between | space-around
align-items: center | flex-start | flex-end | stretch
flex-wrap: wrap; - Permite quebra de linha

Itens Flex (filhos):

flex: 1; - Cresce e encolhe proporcionalmente
align-self: center | flex-start | flex-end
order: 2; - Ordem de exibição

4. CSS Grid - Layout Bidimensional

O que é: Sistema bidimensional para layouts complexos em linhas e colunas.

Container Grid (pai):

display: grid; - Ativa o grid
grid-template-columns: 1fr 1fr 1fr; - Define colunas
grid-template-rows: 100px auto; - Define linhas
gap: 10px; - Espaço entre células
justify-items: center | start | end
align-items: center | start | end

Itens Grid (filhos):

grid-column: 1 / 3; - Ocupa da coluna 1 à 3
grid-row: 1; - Ocupa linha 1
justify-self: center | start | end
align-self: center | start | end

5. Alinhamento de Texto

O que é: Propriedades específicas para alinhar conteúdo textual.

text-align: left | center | right | justify
vertical-align: top | middle | bottom | baseline
line-height: 1.5; - Altura da linha (espaçamento)
text-indent: 20px; - Recuo da primeira linha

6. Margin e Padding - Espaçamento

Diferença: Margin é espaço externo, Padding é espaço interno.

margin: 10px; - 10px em todos os lados
margin: 10px 20px; - 10px top/bottom, 20px left/right
margin: 10px 20px 30px 40px; - top, right, bottom, left
margin: 0 auto; - Centraliza horizontalmente
padding: 15px; - Espaço interno
Margin: 20px auto (centralizado)
Padding: 20px (interno)

📋 Guia Rápido - Quando Usar Cada Método

Para centralizar horizontalmente:

Elementos block: margin: 0 auto;
Elementos inline/inline-block: text-align: center; (no container)
Com Flexbox: justify-content: center;
Com Grid: justify-items: center;

Para centralizar verticalmente:

Com Flexbox: align-items: center;
Com Grid: align-items: center;
Com position: top: 50%; transform: translateY(-50%);
Texto simples: line-height: [altura-do-container];

Para centralizar nos dois eixos:

Flexbox: justify-content: center; align-items: center;
Grid: justify-items: center; align-items: center;
Position: top: 50%; left: 50%; transform: translate(-50%, -50%);
